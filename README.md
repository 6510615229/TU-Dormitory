## 📖 Overview
Finding accommodation near the university can be overwhelming due to scattered information and a lack of centralized data. **TU Dormitory Finder** solves this by providing a unified platform where students can easily search, filter, and view details of off-campus dormitories.

The app focuses on an **intuitive UI/UX** similar to leading hotel booking platforms, ensuring a familiar and seamless experience for users.

## ✨ Key Features

*   **🔍 Smart Filtering System:**
    *   Filter dorms by **Price Range**, **Amenities** (e.g., Air conditioning, Furniture), and **Distance** from the campus.
*   **📱 User-Friendly Interface:**
    *   Designed with a clean, modern UI inspired by industry-standard booking apps to minimize the learning curve.
*   **🔥 Real-time Database:**
    *   Powered by **Cloud Firestore** to store and retrieve dormitory data instantly.
*   **📊 Data Analytics:**
    *   Collects user interaction data (anonymous) to analyze preferences and improve search relevance.

## 🛠 Tech Stack

*   **Mobile Framework:** [Dart, Flutter, Android Studio, Firebase (Firestore)]
*   **Backend / Database:** Google Cloud Firestore (Firebase)
*   **Version Control:** Git & GitHub
*   **Design:** [Figma]

## 🚀 How to Run

1.  **Clone the repository**
    ```
    git clone https://github.com/6510615229/TU-Dormitory
    cd TU-Dormitory
    ```
2.  **Install dependencies**
    ```bash
    flutter clean
    flutter pup get
    ```
3.  **Run the App**
    ```bash
    flutter run
    ```

Alternatively, you can download the APK file to install directly on your Android device via this [Link](https://drive.google.com/file/d/19zW9Wu0H3_olZaPVPO02Js1rbRy7TVrR/view?usp=sharing)
